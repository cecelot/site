### Hardware

- 16" [MacBook Pro](https://www.apple.com/macbook-pro/) w/M3 Pro chip and 36GB memory and 1TB disk
- [iPhone](https://www.apple.com/iphone) 14 Pro
- [AirPods Pro](https://www.apple.com/airpods-pro/) (2nd generation)
- [Apple Watch](https://www.apple.com/watch) SE (1st generation)

### Primary Dev Tools

- [Visual Studio Code](https://code.visualstudio.com) with [direnv](https://direnv.net) and [Nix](https://nixos.org) for project-specific environments
- [iTerm2](https://iterm2.com/) as terminal emulator
- [OrbStack](https://orbstack.dev) for using Docker and running headless virtual machines

### Other Dev Tools

- [IntelliJ IDEA Ultimate](https://www.jetbrains.com/idea/) for Java
- [Zed](https://zed.dev) for keeping an eye on the VSCode competition
- [UTM](https://mac.getutm.app) for graphical virtual machines

### Utilities

- [Tailscale](https://tailscale.com) for private connections to my home server from other networks
- [Mos](https://github.com/Caldis/Mos) for configuring my MX Master 3S settings
- [Loop](https://github.com/MrKai77/Loop) for window management
- [MonitorControl](https://github.com/MonitorControl/MonitorControl) for managing the brightness of my external display

### Creative Tools

- [Affinity Photo 2](https://affinity.serif.com/en-us/photo/) for editing photos
