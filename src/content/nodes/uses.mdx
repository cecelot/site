### Hardware

- 16" [MacBook Pro](https://www.apple.com/macbook-pro/) w/M3 Pro chip and 36GB memory and 1TB disk
- [iPhone](https://www.apple.com/iphone) 14 Pro
- [AirPods Pro](https://www.apple.com/airpods-pro/) (2nd generation)
- [Apple Watch](https://www.apple.com/watch) SE (1st generation)
- [Logitech MX Master 3S](https://www.logitech.com/en-us/shop/p/mx-master-3s)

### Primary Dev Tools

- [Visual Studio Code](https://code.visualstudio.com) with [direnv](https://direnv.net) and [Nix](https://nixos.org) for project-specific environments
- [iTerm2](https://iterm2.com/) as terminal emulator
- [OrbStack](https://orbstack.dev) for using Docker and running headless virtual machines

### Other Dev Tools

- [IntelliJ IDEA Ultimate](https://www.jetbrains.com/idea/) for Java
- [Zed](https://zed.dev) for keeping an eye on the VSCode competition
- [UTM](https://mac.getutm.app) for graphical virtual machines

### Utilities

- [Tailscale](https://tailscale.com) for private connections to my home server from other networks
- [Mos](https://github.com/Caldis/Mos) for configuring my MX Master 3S settings
- [Loop](https://github.com/MrKai77/Loop) for window management
- [MonitorControl](https://github.com/MonitorControl/MonitorControl) for managing the brightness of my external display

### Creative Tools

- [Affinity Photo 2](https://affinity.serif.com/en-us/photo/) for editing photos
- [Affinity Publisher 2](https://affinity.serif.com/en-us/publisher/) for designing beautiful documents

The Affinity suite is pretty spectacular, especially for only being a one-time purchase.

### Browsers

Mainly Safari, but I also have these installed:

- [Chrome](https://google.com/chrome/) for when I need something Chromium
- [Dia](https://diabrowser.com) because I like seeing what [The Browser Company](https://thebrowser.company) is up to

### Research

- [Zotero](https://www.zotero.org) for keeping track of papers

### Style

- [Catppuccin](https://catppuccin.com) as my color scheme of choice
  (including this site!)
- **Fonts**:
  - [Recursive](https://www.recursive.design) for monospace text in my terminal emulator, code editors, etc.
  - [Lora](https://fonts.google.com/specimen/Lora) for serif text and [IBM Plex Sans](https://fonts.google.com/specimen/IBM+Plex+Sans) for sans-serif text (i.e. on this website)

### In My Projects

- **Languages**:
  - [Rust](https://www.rust-lang.org) [wherever I can](https://github.com/sydrinea?tab=repositories&q=&type=&language=rust&sort=)
  - [TypeScript](https://www.typescriptlang.org) for websites (along with HTML and CSS)
  - [Python](https://python.org) for small scripts
  - [MDX](https://mdxjs.com) for authoring content (like this uses page!)
- **Libraries/Frameworks**:
  - [Astro](https://astro.build) for building static sites
  - [Next.js](https://nextjs.org) for React applications
  - [Tailwind](https://tailwindcss.com) whenever I need to write CSS
  - [React](https://react.dev) components for interactivity
  - [Axum](https://github.com/tokio-rs/axum) for web servers
- **Tools**:
  - [Cloudflare](https://www.cloudflare.com) for registrar, DNS, object storage, image optimization, and more
  - [Vercel](https://vercel.com) and [GitHub Pages](https://pages.github.com) for website hosting
  - [PostgreSQL](https://postgres.org) as SQL database of choice
